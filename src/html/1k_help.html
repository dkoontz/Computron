The Computron is made up of several locations, each of which can hold a single integer value. There are 4 general purpose locations (triangle, square, hexagon, star) and 4 special purpose locations (input_a, input_b, output, result). input_a and input_b are used to represent values being fed into the program. output is where the final result of the program is placed before the program ends. result is updated with the result of mathmatical operations.


A Computron program is made up of one or more commands, each on its own line. There are several commands available, the basic commands are:

copy <source location> <destination location>

Copies the contents of location <source> to location <destination>

Given the following state:
Triangle: 7
Square: 2

And the following command: copy triangle square
the result will be:
Triangle: 7
Square: 7

All locations are valid as both sources and destinations.

copy <value> <destination location>

Copies the literal value to the location.
Given the following state:
Triangle: 7

And the following command: copy 3 triangle
the result will be:
Triangle: 3

add
Adds the contents of triangle and square, placing the sum into result.

subtract
Subtracts the contents of square from the contents of triangle and places the difference into result.

multiply
Multiplies the contents of triangle and square, placing the product into result.

divide
Divides the contents of triangle by the contents of square, placing the quotient into result.

divide_full
Divides the contents of triangle by the contents of square, placing the quotient into result and the remainder into star.

compare <shape location> <shape location>
Compares the contents of two shape locations. The result is stored for use in a subsequent jump_if instruction.

:<label>
Creates a label, which itself doesn't do anything. Labels are used as a reference point for jump commands. See the jump command for a usage example.

jump <label>
Moves the control of the program to the label speciﬁed.

copy 1 triangle
copy triangle square
jump :A <-- program is here
add
:A

copy 1 triangle
copy triangle square
jump :A
add <-- skipped
:A <-- program is now here

jump_if_less <label>
Moves control of the program to the label speciﬁed if the last compare resulted in the ﬁrst location being smaller than the second location.

jump_if_greater <label>
Moves control of the program to the label speciﬁed if the last compare resulted in the ﬁrst location being larger than the second location.

jump_if_equal <label>
Moves control of the program to the label speciﬁed if the last compare resulted in the ﬁrst location being equal to the second location.

alias <original_name> <new_name>
Defines a new name for a location. The original name can still be used interchangably with the new name.


Lines you wish the program to ignore can be prefaced with a #

# This line will be ignored
